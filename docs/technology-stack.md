# Production Technology Stack

## Frontend Target

Next.js 15, React, TypeScript, TailwindCSS, ShadCN UI, TanStack Query, Zustand, ECharts, and AG Grid.

The current `apps/web` console is a dependency-free foundation UI so it can run immediately in an empty repository. Its command-center layout and tokens are ready to migrate into the Next.js application boundary.

## Backend

FastAPI, Python, PostgreSQL, TimescaleDB, Redis, RabbitMQ, WebSockets, and gRPC.

## AI

OpenAI or local LLM adapters, LangGraph, CrewAI, embeddings, and pgvector. The Level-1 database migration enables pgvector-backed learning records.

## Trading

MetaTrader 5, MQL5 EA Bridge, Windows VPS Machine Agent, and the MT5 Python package.

## Monitoring And Deployment

Prometheus, Grafana, Loki, OpenTelemetry, Docker, Kubernetes-ready deployment directories, and Windows VPS agent support.
